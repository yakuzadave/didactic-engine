# Quick Start: Marimo Notebook

## Run the Tutorial Notebook

```bash
# Install marimo (if not already installed)
pip install marimo

# Run the tutorial in interactive mode
marimo edit examples/tutorial_marimo.py
```

This will open a web browser with the interactive notebook.

## What You Get

- ✅ **Interactive editing**: Edit code cells and see results instantly
- ✅ **Automatic reactivity**: Cells re-run when dependencies change
- ✅ **Git-friendly**: Pure Python file format with clean diffs
- ✅ **No hidden state**: Guaranteed reproducible execution

## Alternative: Read-Only Mode

Run as an app without editing:

```bash
marimo run examples/tutorial_marimo.py
```
